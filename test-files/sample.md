## Features

The Document Analysis Agent supports:

1. **PDF Files** - Extracted using PDF.js
2. **Text Files** - Direct text reading
3. **CSV Files** - Comma-separated values
4. **Markdown Files** - Formatted text with markup

## Code Example

```javascript
const documentTypes = ['pdf', 'txt', 'csv', 'md'];
console.log('Supported formats:', documentTypes);
```

## Table Example

| Format | Extension | Processing Method |

| Text   | .txt      | FileReader        |
